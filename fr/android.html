Electrum pour Android

Cette page explique comment installer Electrum sur les appareils Android.

Notez qu'Electrum n'est pas distribué sous forme de paquet binaire, mais comme code source Python. Cela signifie que l'installation est légèrement plus compliquée que de télécharger l'application depuis le Play Store.

Il est possible de tout installer à partir des codes QR. Si vous avez des problèmes, vous pouvez trouver de l'aide en suivant ce lien.

1. Téléchargez et installez Google Scripting Layer for Android

Vous pouvez le récupérer ici, ou en scannant le code QR suivant :

2. Téléchargez et installez Python for Android

Vous pouvez le récupérer ici, ou en scannant le code QR suivant :



Une fois que vous avez installé l'APK, lancez l'application Python for Android et cliquez sur 'Install'

3. Téléchargez le script d'installation d'Electrum

1. Lancez SL4A. 2. Appuyez sur le bouton 'Menu'. 3. Appuyez sur 'Add'. 4. Appuyez sur 'Scan Barcode'. 5. Scannez le code QR suivant :


Cela installera un script nommé e4a_install.py

4. Téléchargez et installez Electrum

1. Appuyez sur e4a_install.py : le script téléchargera et installera un dossier nommé "e4a-1.8.1" 2. Pour lancer Electrum, allez dans le dossier "e4a-1.8.1" et appuyez sur 'e4a.py'

5. Créez un raccourci sur votre écran d'accueil

1. Allez sur votre écran d'accueil, appuyez sur le bouton Menu et sélectionnez "Ajouter à l'écran d'accueil". 2. Dans le menu qui s'affiche, choisissez "Raccourci", puis "Scripts" 3. Naviguez vers le script e4a.py.
