Warum Electrum?

  • Sofort Einsatzbereit: Electrum benötigt keine lokale Kopie der Blockchain, diese liegt auf externen Servern.
  • Fehlertolerant: Ihre Geldbörse kann aus Ihren geheimen Startwerten (Seed) wiederhergestellt werden.
  • Sicher: Ihr Seed und private Schlüssel werden nie zum Server geschickt. Vom Server empfangene Daten werden mittels SPV verifiziert.
  • Keine Ausfälle: Mehrere öffentliche Server stellen sicher, dass Electrum durchgängig funktioniert.
  • Universell: Sie können die selbe Geldbörse auf mehreren Geräten verwenden, sie wird automatisch synchronisiert.
  • Offen: Private Schlüssel können in andere Clients importiert werden.
  • Geprüft und getestet: Electrum ist Open Source und existiert seit November 2011.

Erste Schritte

  • Sobald Electrum heruntergeladen und zum ersten Mal gestartet wurde, werden Sie aufgefordert, eine Geldbörse zu erstellen.
  • Wählen Sie einen beliebigen Server aus der Liste und klicken Sie "Ok".
  • Ein Zufallsstartwert (Seed) wird für Sie erstellt. Bewahren Sie diesen sicher auf, er ermöglicht die Wiederherstellung der Geldbörse bei Verlust.
  • Es wird empfohlen, ein sicheres Passwort zum Verschlüsseln Ihrer Geldbörse einzurichten.
  • Sie können nun Electrum zum Senden und Empfangen von Bitcoins verwenden.
  • Neue Adressen werden automatisch generiert, wenn Ihre Empfangsadressen aufgebraucht sind.

Geldbörse aus Seed wiederherstellen / Geldbörse ohne Seed erstellen

  • Falls Sie Ihre Geldbörse verloren haben, können Sie diese aus ihrem Seed wiederherstellen.
  • Starten Sie dazu Electrum und wählen Sie den Punkt "Wiederherstellen", geben Sie den Seed ein und lassen Sie "gap limit" auf der Standardeinstellung, wenn Sie es im vorherigen Electrum nicht geändert haben.
  • Folgen Sie den Schritten als ob Sie zum ersten Mal Ihre Geldbörse erstellen.
  • Sehen Sie sich das Tutorial "Backups erstellen" an, um zu lernen wie Sie auch Ihre Kontenbezeichnungen sichern können, für den Fall dass Sie jemals erneut Ihre Geldbörse wiederherstellen müssen.

  • Geldbörsen ohne Seed können Adressen zum Empfang von Bitcoins erzeugen, jedoch keine senden.
  • Um eine Geldbörse ohne Seed zu erstellen, starten Sie Electrum, wählen Sie "Wiederherstellen" und geben Sie anstelle des Seed Ihren öffentlichen Hauptschlüssel ein.
  • Ein Anwendungszweck für Geldbörsen ohne Seed sind Point Of Sale-Systeme.

Offline-Transaktionen mittels des Öffentlichen Hauptschlüssels erstellen

Sie benötigen einen 2. PC der offline ist. Dies kann entweder ein physisches Gerät oder eine separate Installation auf Ihrem jetzigen Computer sein. Wir empfehlen für besondere Sicherheit eine Debian-Installation mit verschlüsseltem Home-Ordner.

  • [Offline PC] Electrum via USB-Stick installieren.
  • [Offline PC] Neue Geldbörse erstellen. Seed aufschreiben und einprägen, anschliessend Seed vernichten oder in einem Schliessfach sicher verwahren.
  • [Offline PC] Zu Einstellungen -> Import/Export gehen, "Öffentlichen Hauptschlüssel" kopieren und in Text-Datei auf USB-Stick sichern.
  • [Online PC] Electrum installieren und beim ersten Start Wiederherstellen auswählen, "Öffentlichen Hauptschlüssel" eingeben.
  • [Online PC mit bestehender Electrum-Installation] Lesen Sie weiter unten wie Sie alternative oder non-standard Geldbörsen öffnen oder wiederherstellen können.

Sie haben nun eine Geldbörse mit Internetverbindung die Ihnen erlaubt Adressen herauszugeben und Guthabenstände zu prüfen, jedoch ohne die Möglichkeit Coins auszugeben. Sollte es nun einem Angreifer gelingen Ihren am Internet angeschlossenen Rechner zu übernehmen, können Ihre Coins nicht verloren gehen.

Um eine Transaktion auszulösen:

  • [Online PC] Gehen Sie zum "Senden"-Tab und erstellen Sie eine Transaktion. Statt diese zu senden, wird Electrum eine Geldbörse ohne Seed erkennen und nach einem Speicherort für die Transaktion fragen. Wählen Sie Ihren USB-Stick.
  • [Offline PC] Einstellungen -> Import/Export -> "Roh-Transaktion laden". Transaktion vom USB-Stick auswählen. Es wird erkennen daß diese nicht signiert ist, und Sie auffordern dies zu tun. Geben Sie Ihr Passwort ein und signieren Sie die Transaktion. Speichern Sie die neue, signierte, Transaktion auf dem USB-Stick.
  • [Online PC] Einstellungen -> Import/Export -> "Roh-Transaktion Laden". Signierte Transaktion auswählen, und Sie werden gefragt ob Sie diese senden möchten.

Was mache Ich mit meinen alten Adressen und Geldbörsen wenn ich zu Electrum wechsle?

Der beste Weg zu Electrum zu wechseln ist, alle Bitcoins in Ihrer alten Geldbörse an eine neue Adresse in Ihrer Electrum-Geldbörse zu senden. Dadurch sind all Ihre Bitcoins durch Ihren Seed gesichert.

Sie könnten Ihre privaten Schlüssel des alten Clients importieren, dafür müssen Sie jedoch ein Backup der alten Keys aufbewahren, da diese NICHT Teil des Electrum-Seeds werden.

Wechseln, Erstellen oder Wiederherstellen mehrerer Geldbörsen

Es könnte sein, dass Sie eine andere Geldbörse öffnen oder wiederherstellen möchten. Dies ist möglich durch Start von Electrum über die Befehlszeile mit Angabe der zu öffnenden Geldbörse. Wenn diese noch nicht existiert, so wird sie erstellt oder wiederhergestellt.

Beispiel (für OSX):

cd /Applications/Electrum.app/Contents/MacOS
./Electrum -w ~/path/to/your-new-wallet-file

Um zu ihrer Standard-Geldbörse zurückzukehren starten Sie Electrum ohne -w Parameter (Oder einfach über das Programm-Icon starten).

cd /Applications/Electrum.app/Contents/MacOS
./Electrum

Sobald mehrere Geldbörsen existieren können Sie zwischen diesen über "Datei" -> "Geldbörse Öffnen" wechseln.
