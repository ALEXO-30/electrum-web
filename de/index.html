Electrum

Lightweight Bitcoin Client


Electrum ist ein einfach zu bedienender Bitcoin-Client. Er schützt Ihre Bitcoins vor Datenverlust, da Ihre Geldbörse aus einer geheimen Wortfolge wiederhergestellt werden kann, die Sie aufschreiben oder auswendig lernen können. Electrum startet sofort, da keine Kopie der Blockchain heruntergeladen wird.


Warum Electrum?



Features:

  • Sofort Einsatzbereit: Electrum benötigt keine lokale Kopie der Blockchain, diese liegt auf externen Servern.
  • Fehlertolerant: Ihre Geldbörse kann aus Ihren geheimen Startwerten (Seed) wiederhergestellt werden.
  • Sicher: Ihr Seed und private Schlüssel werden nie zum Server geschickt. Vom Server empfangene Daten werden mittels SPV verifiziert.
  • Keine Ausfälle: Mehrere öffentliche Server stellen sicher, dass Electrum durchgängig funktioniert.
  • Universell: Sie können die selbe Geldbörse auf mehreren Geräten verwenden, sie wird automatisch synchronisiert.
  • Offen: Private Schlüssel können in andere Clients importiert werden.
  • Geprüft und getestet: Electrum ist Open Source und existiert seit November 2011.

Benutzeroberflächen

Electrum hat mehrere Oberflächen, die auf dem selben zugrundeliegenden Code basieren: Classic (Qt), Lite, Gtk, Android, Text (mittels curses). Beispiele finden Sie unter Screenshots.
