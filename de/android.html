Electrum für Android

Diese Seite erklärt die Installation von Electrum auf Android-Geräten.

Beachten Sie bitte, dass Electrum nicht als Paket, sondern als Python-Quellcode verteilt wird. Die Installation ist daher geringfügig komplizierter als ein App aus dem Play Store herunterzuladen.

Es ist aber möglich, alles über QR-Codes zu installieren. Bei Schwierigkeiten können Sie unter diesem Link Hilfe suchen..

2. Python für Android installieren

Gibt es hier, oder den folgenden QR-Code einscannen:



Sobald das APK installiert ist, starten Sie "Python für Android" und klicken "installieren".

3. Electrum-Installationsskript herunterladen

1. SL4A starten
2. Menü-Button drücken
3. "Add" antippen
4. "Scan Barcode" antippen
5. Den folgenden QR-Code scannen:


Dies installiert ein Skript namens e4a_install.py

4. Electrum laden und installieren

1. e4a_install.py antippen: dies lädt und installiert ein Verzeichnis namens "e4a-1.8.1"
2. Um Electrum zu starten, dieses Verzeichnis öffnen und "e4a.py" antippen

5. Einen Shortcut auf dem Homescreen einrichten

1. Zum Homescreen gehen, "Menü" anklicken and "Zu Homescreen hinzufügen" auswählen.
2. Im angezeigten Menü "Verknüpfung" auswählen, dann "Skripte"
3. Zum e4a.py-Skript navigieren.
