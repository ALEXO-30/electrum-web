    Ubuntu / Debian
    (version 1.5.6)
    • Add a new repository:
      sudo add-apt-repository "deb http://apt.bitcoins.sk/ precise main"
    • Add the repo key to apt:
      wget -O - http://apt.bitcoins.sk/deafboy@bitcoins.sk.pub | sudo apt-key add -
    • Update index:
      sudo apt-get update
    • Install Electrum:
      sudo apt-get install electrum
    • Alternatively download .deb directly:
      electrum_1.5.6.4_all.deb
    • Install Electrum:
      sudo dpkg -i electrum_1.5.6.4_all.deb

    Development version

    Dependencies: python, qt, pyqt, gettext, ecdsa, slowaes. On Ubuntu you can install the dependencies with:

      sudo apt-get install python-pip python-qt4 pyqt4-dev-tools
  sudo pip install ecdsa slowaes

    Check out the code from Github

      git clone git://github.com/spesmilo/electrum.git

    Compile the icons

      pyrcc4 icons.qrc -o lib/icons_rc.py

    Create translations

      python mki18n.py

    Install Electrum

      sudo python setup.py install
