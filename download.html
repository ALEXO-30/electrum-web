Building from source

Dependencies

  • Python
  • qt
  • pyqt
  • gettext
  • ecdsa
  • slowaes

On Ubuntu you can install the deps with.

sudo apt-get install python-pip python-qt4 pyqt4-dev-tools
sudo pip install ecdsa slowaes

Check out the code from Github

  git clone git://github.com/spesmilo/electrum.git

Compile the icons

  pyrcc4 icons.qrc -o lib/icons_rc.py

Create translations

  python mki18n.py

Install Electrum

sudo python setup.py install
electrum

Source on Windows

  • Install PyQt4
  • Install Electrum-1.0.zip
  • execute 'python electrum'

Source on OS X

  • Install Homebrew.
  • Install depedencies brew install qt pyqt gettext pip.
  • Install Python dependencies pip install ecdsa slowaes.
  • Follow Linux build instructions.
