Sources Electrum-1.5.6.tar.gz or Electrum-1.5.6.zip
Linux
  • Install python-qt4 and pip:
    sudo apt-get install python-qt4 python-pip
  • Install electrum:
    sudo pip install https://github.com/downloads/spesmilo/electrum/Electrum-1.5.6.tar.gz
  • Run:
    electrum

      sudo apt-get install electrum

    Development version

    Dependencies: python, qt, pyqt, gettext, ecdsa, slowaes. On Ubuntu you can install the dependencies with:

      sudo apt-get install python-pip python-qt4 pyqt4-dev-tools
  sudo pip install ecdsa slowaes

    Check out the code from Github

      git clone git://github.com/spesmilo/electrum.git

    Compile the icons
