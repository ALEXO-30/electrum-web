Download

Stable release

Ubuntu Add this to your /etc/apt/sources.list:
deb http://apt.bitcoins.sk/ precise main
Add the key to apt:
wget -O - http://apt.bitcoins.sk/deafboy@bitcoins.sk.pub | sudo apt-key add -
Install Electrum:
sudo apt-get update
sudo apt-get install electrum
Linux In a terminal, type:
sudo pip install https://github.com/spesmilo/electrum/tarball/1.0\#md5\=863e61b215bf2e3fee243ce9583f677f
Then, to start Electrum, type electrum

Building from source

Dependencies

  • Python
  • qt
  • pyqt
  • gettext
  • ecdsa
  • slowaes

On Ubuntu you can install the deps with.

sudo apt-get install python-pip python-qt4 pyqt4-dev-tools
sudo pip install ecdsa slowaes

Check out the code from Github

  git clone git://github.com/spesmilo/electrum.git

Compile the icons

  pyrcc4 icons.qrc -o lib/icons_rc.py

Create translations

  python mki18n.py

Install Electrum

sudo python setup.py install
electrum

Source on Windows

  • Install PyQt4
  • Install Electrum-1.0.zip
  • execute 'python electrum'
