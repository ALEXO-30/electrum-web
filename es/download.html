    Versión de desarrollo

    Dependencias: python, qt, pyqt, gettext, ecdsa, slowaes. En Ubuntu puedes instalar las dependencias con:

    		  sudo apt-get install python-pip python-qt4 pyqt4-dev-tools
		  sudo pip install ecdsa slowaes

    Copia el código desde Github

    		  git clone git://github.com/spesmilo/electrum.git

    Compila los iconos

    		  pyrcc4 icons.qrc -o gui/icons_rc.py

    Crea las traducciones

    		  python mki18n.py

    Instala Electrum

    		  sudo python setup.py install
